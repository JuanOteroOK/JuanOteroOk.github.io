Información

Este proyecto para su FrontEnd esta realizado con HTML, CSS, Bootstrap, FontAwesome, en algunas vistas JQuery y scripts de JavaScript para algunas funcionalidades. En la parte BackEnd el proyecto esta realizado en PHP puro y para la base de datos se uso MySQL. El sistema cuenta con dos grandes modulos, uno para el cliente, que tiene funcionalidades como: registro de clientes, login de clientes, carrito de compras para clientes, editar productos dentro del carrito ya sea cantidad de productos o eliminar productos, barra de busqueda de productos, vista de los productos, vista de los productos en oferta, vista para ver las compras realizadas por el cliente en esta misma con detalles como fecha, monto y estado de la compra, y opcion para efectuar el pago de sus compras. Por otro lado esta el modulo del administrador de la web, que tiene funcionalidades como: Logearse como Admin, Agregar productos con todos sus detalles, Agregar Categoria de Productos, Administrar las ofetas de los productos que desee, Se visualizan pintados los productos con stock menor a un cierto numero para reponerlos nuevamente, Manejar el Tracking de los productos vendidos, Ver los pagos de los clientes ya sean los efectuados o los pendientes. Entre otros detalles.
