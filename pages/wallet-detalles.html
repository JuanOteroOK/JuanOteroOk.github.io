Información

Este proyecto fue un requisito para la aprobacion del Programa PIL: Nuevas tecnologias, del gobierno de la Provincia de Cordoba, la aprobacion de este dio lugar a la Certificacion de Desarrollador FullStack, ya que en la capacitacion intensiva de 6 meses, se vieron los temas: Scrum, Comunicacion Efectiva, Git & Github, Angular, TypeScript, C#, SQL Server, CORS, JWT y Postman entre otras cosas. Para su FrontEnd esta realizado con Angular 12, Angular Material, TypeScript, Bootstrap, CSS, AOS y SweetAlert. En la parte BackEnd el proyecto esta realizado en .NET (C# EntityFramework WebApi) con MVC, para la base de datos se uso SQL Server y para la seguridad de la app se utilizo Jason Web Token (JWT) como generador de token y validador. El sistema cuenta con modulos para el cliente, que tienen funcionalidades como: Registro de Clientes, Login de Clientes, Operaciones para el Cliente (Deposito, Extraccion y Transferencia), Ver Historial de Operaciones del Cliente, Ver y Editar informacion personal de la Cuenta del Cliente. Entre otros detalles y componentes.
